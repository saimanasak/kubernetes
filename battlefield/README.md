## Kubernetes Battlefield ⚔️

Welcome to the **Kubernetes Battlefield** - a hands-on learning space where different Kubernetes concepts are explored through practical projects.

Each project focuses on a specific Kubernetes component or workflow, helping build a deeper understanding of how applications are deployed, managed, and exposed inside a Kubernetes cluster.

The goal of this repository is to learn Kubernetes by **building, experimenting, and troubleshooting real scenarios**, just like a battlefield where every concept is tested in action.

---

#### Each project contains:

- Step-by-step implementation
- Architecture diagrams
- Blog explanations
- Source manifests or Helm charts

#### Learning Approach

Instead of only reading documentation, this repository focuses on **learning Kubernetes through practical experimentation**.

Each project builds on the previous one, gradually introducing new Kubernetes components and real-world deployment practices.
